# Prompt: final polish without scope expansion

```md
Сделай только финальную полировку уже реализованного экрана.

Контекст:
- Основной экран уже собран по этапам.
- Нужно улучшить качество без расширения scope.

Scope:
- Проверить адаптивность.
- Проверить keyboard/accessibility basics.
- Подправить spacing, alignment, text overflow.
- Упростить локальные шероховатости в коде, если они появились во время итераций.

Out of scope:
- Не добавлять новый функционал.
- Не внедрять реальный API.
- Не менять архитектуру страницы.
- Не делать глобальный refactor.

Acceptance:
- Экран корректно выглядит на desktop/tablet/mobile.
- Не сломаны состояния loading/empty/error.
- Не сломаны filters/tabs/selection.
- Нет лишних визуальных скачков layout.
- Изменения ограничены polish-слоем.

После реализации:
- Коротко перечисли, что было исправлено.
- Отдельно перечисли остаточные риски.
```
