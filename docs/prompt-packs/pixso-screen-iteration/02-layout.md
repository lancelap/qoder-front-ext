```md
Реализуй только первый этап экрана: каркас страницы и общую структуру.

- Есть отдельные секции под filters, tabs и table.
- Layout адаптивен на базовом уровне.
- Код компилируется в текущем TS-проекте.
- Изменения ограничены layout-слоем.

Constraints:
- Использовать существующие проектные primitives, если они уже есть.
- Не добавлять новые зависимости.
- Не делать рефакторинг соседних файлов без необходимости.

Верни только изменения для этого этапа и коротко перечисли, что осталось на следующие итерации.
```
